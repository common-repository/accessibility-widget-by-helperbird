= Installing the widget =


Include our script below to the top of webpage.

``
<script src="https://assets.helperbird.com/widget/widget.js"></script>
``

Then near the bottom of website, add the following

``
<helperbird-widget></helperbird-widget>
``

Congratulations you now have the Helperbird Widget installed.
